# Prérecquis :

- Node.JS installé
- MySQL installé

# Pour initialiser ce projet :

Installer les dépendances avec la commande suivante :

- npm install
- node firstconnection.js
- configurer le fichier .env avec ses paramètres

# Démarrer le serveur avec la commande suivante :

node server

# 4 routes accessible avec l'API :

- GET /images
- GET /images/:id
- POST /images
- DELETE /images/:id
